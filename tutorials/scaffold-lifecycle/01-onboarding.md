# Chapter 1: Onboarding

## What Happens

The `/onboard` skill asks you about your project. For this tutorial, here's what to tell it:

- **What are you building?** A trading app that connects to the Mangrove developer API for marketplace queries, DEX swaps, and wallet management.
- **Why?** Learning the app-in-a-box lifecycle (or building a real trading tool — your call).
- **Experience level:** Your actual level.
- **Project name:** my-trading-app (or whatever you prefer)

## Expected Output

After onboarding:
- `branding.json` updated with your project name
- `CLAUDE.md` updated with project context
- Ready to run `/requirements`
